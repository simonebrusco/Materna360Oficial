# P34.ADM.1 — Evidências (Quick Ideas via ADM)

## Contexto
Esta P consolida o ADM como **fonte primária de conteúdo** para Quick Ideas.  
A IA deixa de gerar conteúdo e passa a **orquestrar, filtrar e selecionar**
ideias previamente curadas e publicadas no Supabase (`adm_ideas`).

---

## Endpoint testado
`POST /api/ai/quick-ideas`

---

## Teste 1 — hub = my_day (admHub = meu-dia-leve)

### Request (curl)
```bash
curl -s http://localhost:3001/api/ai/quick-ideas \
  -H "Content-Type: application/json" \
  --data '{"intent":"quick_idea","hub":"my_day","nonce":111,"locale":"pt-BR","memory":{"recent_suggestion_ids":[],"last_signal":"neutral"}}'

{"suggestions":[{"id":"mdl-foto-memoria","title":"Foto-memória","description":"Relembrar algo bom sem esforço e com afeto."}],"meta":{"mode":"my_day_light","source":"adm","admHub":"meu-dia-leve"}}

curl -s http://localhost:3001/api/ai/quick-ideas \
  -H "Content-Type: application/json" \
  --data '{"intent":"quick_idea","hub":"cuidar_de_mim","nonce":222,"locale":"pt-BR"}'

{"suggestions":[{"id":"cdm-lavar-rosto","title":"Lavar o rosto com intenção","description":"Um reset físico e mental rapidinho."}],"meta":{"mode":"my_day_light","source":"adm","admHub":"cuidar-de-mim"}}
